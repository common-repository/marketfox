== Description ==

Create fully customizable landing pages, in-app messages, web-forms with drag and drop designer and zero coding. Target specific users based on their behavior and activities in website. Responsive templates to personalise in any platform. 

To get started:
1) <a href="https://www.marketfox.io/">Sign up for a Marketfox account</a> to get the API credentials 
2) Go to your Marketfox plugin settings page and save your information.
